最近在 github 上發現一個功能，不確定是什麼時候開始有的

就是只要建立一個跟自己帳號同名的 repository

裡面 README.md 的內容就會顯示在 profile 上面


### 建立新的倉庫

輸入自己的帳號，github 會跟你說你發現了一秘密
